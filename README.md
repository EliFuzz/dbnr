# Enterprise FuulStack MERN App
___

### Start
```console
make up
```

### Stop
```console
make down
```

### Clean
```console
make clean
```

## Usage
1. `make up`
2. Go to:
   - Backend: [http://localhost:3000/](http://localhost:3000/)
   - Frontend: [http://localhost:3001/](http://localhost:3001/)
